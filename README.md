# ship_alien
用pygame，做一个飞机大战外星船的游戏，运行alien_invasion.py即可（前提是安装好pygame等库）
顶部分别显示当前生命数、当前最高分、史上最高分和关卡数，空格发射子弹，飞船可上下左右移动
